My Work in the Web!

Prework Study Guide

This project, my rite of passage “Hello World,” is a study guide I created while being introduced to the basics of HTML, CSS, and Javascript. While this project is rough around the edges, it displays a very basic understanding and learning of web design. Fundamental concepts such as containers are deployed here to contain our sections. Very simple styling is used on our containers through the use of our property “box-shadow.” While no Javascript is displayed on our page, a quick look in the console via our Chrome Dev Tools will show a few “console.logs.” A brief view of the javascript file will further show a for loop, conditional logic, and the use of a function. Nothing is rather impressive here. However, this project was the perfect start to our journey in the web!
